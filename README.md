# Asset Class Performance Tracker
A Power BI dashboard comparing five year returns across major asset classes including equities, bonds, real estate and infrastructure from 2021 to 2026.

# What it covers
Revenue growth from €161bn to €179.8bn over five years 
Cumulative return comparison across four asset classes over five years

SPY (S&P 500) representing equities
AGG representing bonds and fixed income
VNQ representing real estate investment trusts
IGF representing global infrastructure

# Data Source
Historical price data downloaded from Yahoo Finance and Stooq for all four asset class ETFs from 2021 to 2026

# Tools Used
Power BI Desktop for interactive dashboard and trend visualisation
Microsoft Excel for data structuring, return calculations and percentage change analysis

# Key Insights
Equities (SPY) significantly outperformed all other asset classes over the five year period
Real estate (VNQ) and infrastructure (IGF) showed moderate growth with higher volatility
Bonds (AGG) underperformed across the period reflecting the rising interest rate environment
